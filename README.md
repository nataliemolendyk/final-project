## What does this project do?
This project loads pets and their information from the RescueGroups API as well as the shelter's information, so users can connect if they are interested. This website also provides information about the adoption process, common questions and answers, and advice for new pet owners or anyone looking to learn more about adopting. This website provides thousands of pets (dogs, cats, rabbits, and birds) available for adoption and includes if the animals have siblings that they have lived with, to help make the transition to a new home easier. Lastly, the get in touch page has a number of partner shelters available to contact.
